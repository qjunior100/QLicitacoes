Dúvidas

Perguntas frequentes

O Windows exibe um aviso de segurança ao abrir o programa. É seguro?
Sim, é completamente seguro. O aviso do Windows SmartScreen aparece porque o instalador ainda não possui assinatura digital — um certificado que exige investimento anual e está no nosso roadmap. Para prosseguir: clique em "Mais informações" e depois em "Executar assim mesmo". O QLicitações é um software legítimo que consulta apenas a API pública do governo (PNCP).
O QLicitações funciona em Mac ou Linux?
Por enquanto o QLicitações é exclusivo para Windows 10 ou superior. Versões para outros sistemas estão em nosso roadmap.
De onde vêm os dados das licitações?
Todos os dados são obtidos diretamente do PNCP — Portal Nacional de Contratações Públicas, a base oficial do governo federal. Os dados são sempre atualizados e confiáveis.
Posso cancelar a assinatura a qualquer momento?
Sim. Não há fidelidade ou multa. Você cancela quando quiser diretamente pelo Hotmart e a licença permanece ativa até o final do período pago.
Posso usar em mais de um computador?
Cada chave de licença é vinculada a um computador. Para usar em outra máquina entre em contato com nosso suporte e faremos a transferência.
O que acontece quando minha assinatura vence?
Para assinaturas mensais o Hotmart renova automaticamente. Se o pagamento não for aprovado a licença é suspensa e reativada assim que regularizado.
Posso trocar de segmento no Plano Básico?
O Plano Básico é fixo para o segmento escolhido na compra. Para acesso a múltiplos segmentos recomendamos o Plano Completo.
